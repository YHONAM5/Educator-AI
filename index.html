Acerca de MentorFlexAI
Sistema de Tutoría Educativa Adaptativa

MentorFlexAI es un sistema de tutoría educativa basado en IA que se adapta automáticamente al nivel del estudiante, proporcionando respuestas personalizadas.

APIs Disponibles:
POST
/ask
Respuesta completa con clasificación automática
POST
/classify-level
Clasificación de nivel únicamente
POST
/ask-specific
Búsqueda específica por documento
Características principales:
  • Clasificación automática del nivel del estudiante (básico, intermedio, avanzado)
  • Recuperación de contexto relevante desde documentos educativos
  • Generación de respuestas adaptadas al nivel del estudiante
  • Búsqueda específica en documentos particulares
  • Interfaz de API completa para integración con otras plataformas
Tecnologías utilizadas:
  • Modelo LLM: CodeLlama 7B (Instruct)
  • Embeddings: Sentence-Transformers (all-MiniLM-L6-v2)
  • Base de datos vectorial: ChromaDB
  • Backend: FastAPI
  • Frontend: HTML, CSS, JavaScript
